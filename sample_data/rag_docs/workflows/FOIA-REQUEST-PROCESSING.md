# Workflow example — FOIA request processing *(synthetic)*

## Triage

1. Log request in the case management system with received date.  
2. Assign **complexity tier** (simple, standard, complex) and estimated response date.

## Search and review

3. Identify likely custodians; issue **preservation** instructions if litigation is reasonably anticipated.  
4. Run enterprise search across email and shared drives; deduplicate hits.  
5. Apply **Exemption 5** deliberative process review for pre-decisional drafts; consult legal for **Exemption 6** privacy redactions.

## Response package

6. Prepare Vaughn index for withheld materials when required.  
7. Quality-check redactions (remove hidden revision metadata).  
8. Release responsive records via secure portal or mail; close case with appeal rights language.

## Timeliness

Track **20-business-day** statutory clock; document extensions with written notice citing exceptional circumstances or voluminous records.
